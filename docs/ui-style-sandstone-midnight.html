Suggested light / dark pairs

If you ship user-selectable themes in Settings, these combinations keep the same “family feel” across modes:

Classic pair — Sandstone Classic (light) + Midnight Classic (dark). Warm day, quiet night; different accents but same calm density.
Unified warm — Sandstone Clay (light) + Midnight Ember (dark). Shared amber/terracotta thread; best if you want one brand temperature.
Minimal pair — Sandstone Linen (light) + Midnight Mist (dark). Both strip chrome; only photos and type carry weight.
All Sandstone — Sandstone Classic (light) + Sandstone Night (dark). Same rounded panels and dashed compose in both modes — easiest to implement as one token set with a color-scheme flip.
