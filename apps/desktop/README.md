# Inertia desktop (Tauri)

Native window that starts the local `inertia-api` sidecar and loads the Svelte UI from `http://127.0.0.1:4783`.

App icon source: [`app-icon.png`](./app-icon.png) (midnight mutual-link mark: orange squares + charcoal bridge). Regenerate with:

```bash
cd apps/desktop && npx tauri icon ./app-icon.png
```

See [docs/TAURI.md](../../docs/TAURI.md) for prerequisites and commands.
